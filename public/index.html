Type or Die

http://type-or-die.herokuapp.com/

Type the falling words before they disappear!

This is a web-based typing game that my group created for our first project. There are 3 different modes you play: easy, hard, and insane. We used three different API's to get the words for each mode. We used Firebase to have users login and track their stats. The guts of the application were created with HTML, CSS, Bootstrap, Javascript, and JQuery.

Let's Make a Meal

https://lets-make-a-meal-deal.herokuapp.com/

Let's Make a Meal is a web application that lets users enter in ingredients to find different recipies to make.

This was a group project that used an API call to retrieve different recipes based on the ingredients entered. We stored information into a SQL database. The front end of the site was created using Materialize and custom CSS, HTML, Javascript, and JQuery.

Texas Football Trivia

https://longhorn-trivia.herokuapp.com/

A Texas Longhorn Football Trivia game I created using HTML, CSS, Bootstrap, Javascript, and JQuery.

You have 10 seconds to answer the question. If times runs out or you answer incorrectly, a big X will appear and you will move on to the next question. There are 5 questions total. If you answer correctly you will see a cool photo that resembles the question. Good Luck! And make sure your volume is turned up at the end!

Train Scheduler

https://vast-beach-38934.herokuapp.com/

In this project, I created a train schedule application that incorporates Firebase to host arrival and departure data. My app retrieves and manipulates this information with Moment.js. This website will provide up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station.

TV Show Gif's

https://pure-oasis-11247.herokuapp.com/

In this project, I used the GIPHY API to make a dynamic web page that populates with gifs of your choice. Users can use the pre-populated buttons to look at different gifs for different shows. Users can also add a show to pull gifs from. Once the gif is loaded, all you have to do is click the image in order for it to move.

I used an ajax call to pull data from the Giphy API to pull all the different gifs. I used custom CSS, HTML, and Bootstrap to design the page.
